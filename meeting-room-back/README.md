# meeting-room-back

Projeto Maven com Spring Boot 2.5.3, Java versão 16.

Dependências principais:

* Spring Data JPA;
* Banco de dados H2;
* Lombok.

Endpoints da API na porta configurada:

* GET: ```/sala/{id}``` (sem id: lista completa)
* POST: ```/sala```
* PUT: ```/sala/{id}```
* DELETE: ```/sala/{id}```
